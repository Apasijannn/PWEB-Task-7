Pengenalan HTML Dasar

Mari pahami kerangka dasar dari setiap situs web yang ada di internet sebelum kita melangkah ke desain dan interaktivitas.

Apa itu HTML?

HTML (HyperText Markup Language) bukanlah bahasa pemrograman, melainkan bahasa markup. Tugas utamanya adalah memberikan struktur pada konten web, seperti mendefinisikan mana yang merupakan paragraf, judul, atau gambar.

Contoh Sintaks Dasar:
<!DOCTYPE html>
<html lang="id">
<head>
  <title>Halaman Pertamaku</title>
</head>
<body>
  <h1>Halo Dunia!</h1>
  <p>Ini adalah paragraf pertama saya.</p>
</body>
</html>

Struktur & Tag Meta

Memahami anatomi dokumen HTML dan bagaimana browser serta mesin pencari membaca situsmu di balik layar.

Anatomi Tak Terlihat

Pengguna hanya melihat apa yang ditampilkan di dalam <body>. Namun, "otak" dari sebuah halaman web sebenarnya berada di dalam <head>. Di sinilah kamu mendefinisikan set karakter, pengaturan viewport agar web tidak hancur di layar HP, hingga judul halaman yang muncul di tab browser.

Sintaks Wajib Standar Global:
<!DOCTYPE html>
<html lang="id">
<head>
  <!-- Wajib untuk render teks universal -->
  <meta charset="UTF-8">
  
  <!-- Wajib untuk responsivitas mobile -->
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  
  <title>Dokumen Standar</title>
</head>
<body>
  <p>Konten dimulai di sini.</p>
</body>
</html>

Memahami CSS Selector

Pelajari cara menargetkan elemen HTML secara spesifik untuk diberikan gaya tanpa merusak elemen lainnya.

Tiga Pilar Utama Penargetan

CSS tidak akan bekerja jika ia tidak tahu elemen mana yang harus dihias. Ada tiga cara utama untuk "memanggil" elemen HTML dari file CSS kamu:

  • Element Selector: Menargetkan langsung nama tag (misal: p atau h1). Efeknya akan berlaku ke semua tag tersebut.
  • Class Selector (.): Menargetkan atribut class. Sangat direkomendasikan karena bisa dipakai berulang kali.
  • ID Selector (#): Menargetkan atribut id. Hanya boleh ada satu ID unik dalam satu halaman.
Contoh Implementasi:
/* Menargetkan semua paragraf */
p {
  color: #333333;
}

/* Menargetkan class .tombol-utama */
.tombol-utama {
  background-color: red;
  border-radius: 8px;
}

Konsep Box Model

Rahasia utama di balik tata letak web. Semua elemen yang kamu lihat di browser pada dasarnya adalah sebuah kotak.

Mengapa Desainmu Berantakan?

Pemula sering frustrasi saat elemen HTML saling bertumpuk atau keluar dari batas. Itu terjadi karena mereka tidak memahami Box Model. Setiap kotak elemen terdiri dari empat lapisan (dari dalam ke luar):

  1. Content: Isi utama elemen (teks, gambar).
  2. Padding: Ruang kosong di dalam elemen (jarak antara konten dengan batas).
  3. Border: Garis batas di sekeliling elemen.
  4. Margin: Ruang kosong di luar elemen (jarak elemen ini dengan elemen lainnya).
Reset Box Model Wajib:
/* Mengatur ulang logika perhitungan kotak browser */
* {
  box-sizing: border-box;
  margin: 0;
  padding: 0;
}
